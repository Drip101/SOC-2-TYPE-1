# SOC-2-TYPE-1
SOC 2 / HIPAA / ISO 27001 Simulated Audit Environment

Project Overview

HealthPulse is a U.S.-based healthcare data analytics company that aggregates and analyzes patient healthcare data to improve patient outcomes and reduce operational costs for healthcare providers.

The organization operates primarily in AWS cloud infrastructure and processes Protected Health Information (PHI), requiring compliance with HIPAA regulations. In addition, HealthPulse is pursuing ISO 27001 certification to demonstrate mature information security governance and attract international healthcare clients.

This project simulates a real-world Governance, Risk, and Compliance (GRC) audit engagement focused on:

HIPAA Security Rule compliance
ISO 27001 control alignment
Cloud security governance
Risk management maturity
Technical and administrative safeguards

The assessment evaluates the organization’s people, processes, technologies, and security controls to identify compliance gaps, operational risks, and security improvement opportunities.
